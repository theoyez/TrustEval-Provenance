How it works (Pro)

  1. Run your model on the manifest, write a run manifest with outputs and cited source keys.
  2. Hash cited chunks → build a Merkle root; sign the run manifest (demo signature is simulated).
  3. Export prov/proofs/<run_id>.json and the bundle zip; include the root and signature in metadata.

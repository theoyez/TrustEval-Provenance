How it works

TrustEval shows a leaderboard of runs, per-run reports, and (Pro) provenance proofs. Metrics are precomputed; this site visualizes them.

  • Model pill: run metadata
  • Audit pill: retrieval candidates & citations
  • Provenance pill (Pro): Merkle root & downloadable proof
  • Compare: select two runs to see metric deltas
